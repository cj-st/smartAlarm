tomtom.Marker Class

Extends L.Marker
Defined in: src\tomtom\Marker.js:10

A class which can be used to add an image based marker to the map. Markers are used to highlight points on the map. The marker image can be customized by specifying the iconOptions parameter. Also, a default marker can be set by setting tomtom.Marker.DEFAULT_MARKER_OPTIONS.

Constructor

tomtom.Marker

(
  • latlng
  • [iconOptions]
  • [markerOptions]
)

Defined in src\tomtom\Marker.js:10

Parameters:

  • latlng Array | L.LatLng

    The lat / lng the marker should be added at. Can either pass an array, [lat, lng] or a L.LatLng object.

  • [iconOptions] Object optional

    The options used to build the marker's image. If this is not specified, tomtom.Marker.DEFAULT_MARKER_OPTIONS will be used.

    • iconUrl String

      The URL to the icon image (absolute or relative to your script path).

    • [iconRetinaUrl] String optional

      The URL to a retina sized version of the icon image (absolute or relative to your script path). Used for Retina screen devices.

    • [iconSize] L.Point optional

      Size of the icon image in pixels.

    • [iconAnchor] L.Point optional

      The coordinates of the "tip" of the icon (relative to its top left corner). The icon will be aligned so that this point is at the marker's geographical location. Centered by default if size is specified, also can be set in CSS with negative margins.

    • [shadowUrl] String optional

      The URL to the icon shadow image. If not specified, no shadow image will be created.

    • [shadowRetinaUrl] String optional

      The URL to the retina sized version of the icon shadow image. If not specified, no shadow image will be created. Used for Retina screen devices.

    • [shadowSize] L.Point optional

      Size of the shadow image in pixels.

    • [shadowAnchor] L.Point optional

      The coordinates of the "tip" of the shadow (relative to its top left corner) (the same as iconAnchor if not specified).

    • [popupAnchor] L.Point optional

      The coordinates of the point from which popups will "open", relative to the icon anchor.

    • [className] String optional

      Custom class name to assign to both icon and shadow images. Empty by default.

  • [markerOptions] Object optional

    Additional options to customize the marker.

    • [icon] L.Icon optional

      Icon class to use for rendering the marker. See Icon documentation for details on how to customize the marker icon. Set to new L.Icon.Default() by default.

    • [clickable=true] Boolean optional

      If false, the marker will not emit mouse events and will act as a part of the underlying map.

    • [draggable=false] Boolean optional

      Whether the marker is draggable with mouse/touch or not.

    • [title=''] String optional

      Text for the browser tooltip that appear on marker hover (no tooltip by default).

    • [zIndexOffset=0] Number optional

      By default, marker images zIndex is set automatically based on its latitude. You this option if you want to put the marker on top of all others (or below), specifying a high value like 1000 (or high negative value, respectively).

    • [opacity=1.0] Number optional

      The opacity of the marker.

    • [riseOnHover=false] Boolean optional

      If true, the marker will get on top of others when you hover the mouse over it.

    • [riseOffset=250] Number optional

      The z-index offset used for the riseOnHover feature.

Example:

map.addLayer(new tomtom.Marker([-71.53686, 43.20846]));

DEFAULT_MARKER_OPTIONS

String static

Defined in src\tomtom\Marker.js:360

The default marker options which will be used when no options are specified for the Marker constructor. The default is:

tomtom.Marker.DEFAULT_MARKER_OPTIONS = { url: "marker.png", width: 51, height: 60, offsetWidth: -21, offsetHeight: -49 }

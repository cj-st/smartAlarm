tomtom.controls.ContextMenu Class

Extends L.Control
Defined in: src\tomtom\controls\ContextMenu.js:12

Creates a new ContextMenu which can be used to display a menu to the user when a marker or the map is right clicked.

Constructor

tomtom.controls.ContextMenu

(
  • options
)

Defined in src\tomtom\controls\ContextMenu.js:12

Parameters:

  • options Object

    Options used to build the context menu.

    • menuItems Array

      An array of the menu items to add. Each object should have a label and optionally an onClick callback. See the example below.

Example:

map.setContextMenu(new tomtom.controls.ContextMenu({ menuItems: [ { label: "Test", onClick: function() { alert("TEST!"); } } ] }));

menuitemclick

Defined in src\tomtom\controls\ContextMenu.js:131

An event which is triggered when a menu item is clicked.

Event Payload:

  • eventArgs Object

    Arguments passed to the event that carry information about the event.

    • menuItem Object

      The menu item which was clicked.

    • contextMenu tomtom.ContextMenu

      The context menu that triggered the event.

    • x Number

      The x coordinate of the current map relative position that the menu is being displayed at.

    • y Number

      The y coordinate of the current map relative position that the menu is being displayed at.

    • targetElement DOM Node

      The element for which the menu is being displayed.

    • targetObject Object

      An object associated with the DOM Node for which the menu is being displayed.

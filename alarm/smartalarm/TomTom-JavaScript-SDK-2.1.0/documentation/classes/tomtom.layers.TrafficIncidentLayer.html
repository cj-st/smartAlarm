tomtom.layers.TrafficIncidentLayer Class

Extends L.LayerGroup
Defined in: src\tomtom\layers\TrafficIncidentLayer.js:29

Displays traffic incident icons as a layer above the base map. It is recommended that this class is not used directly but rather the Map's displayTraffic option is used like this:

var map = new tomtom.Map({ domNode: "map", displayTraffic: true });

tomtom.apiKey

Unknown static

Defined in src\tomtom\Main.js:153

Allows the Map Toolkit API key to be set in once place and used by the TomTom Map and services without specifying the key each time a new object is constructed.

Example:

// set a global api key tomtom.apiKey = "API Key";

// the map will use tomtom.apiKey var map = new tomtom.Map({ domNode: "map" });

// same for services var service = new tomtom.services.GeocodingService(); service.geocode("11 Lafayette St, Lebanon, NH", function(response) { ... });

// passing an API Key to a map or service will override the global key. var map2 = new tomtom.Map({ apiKey: "Another API Key", domNode: "map" });

Defined in src\tomtom\services\TrafficService.js:23

Gets traffic incident information for the specified window and zoom. The window should be specified as an object with left, bottom, right, and top properties.

Parameters:

  • window Object

    The bounds in which to retrieve the traffic information.

  • zoom Number

    The zoom level at which to retrieve the traffic information.

  • [options] Object optional

    Additional options to be passed to the traffic call.

    • [language=en] String optional

      The two-character ISO code for the output language. Affects the &<c&> (cause) and &<d&> (description) fields. If this is not specified, those fields will not be included in the reply. Supported languages are ca, cs, da, de, en, es, fi, fr, hu, it, nl, no, pl, pt, sk, sv, tr.

    • [projection=EPSG4326] String optional

      The projection used to specify the coordinates in the request and response.

  • callback Object

Example:

var window = { left: -78.40152658903608, bottom: 39.19651962955681, right: -59.054602760910434, top: 44.89325482670453 }

var trafficService = new tomtom.services.TrafficService("API Key"); trafficService.getTrafficModel(window, 7, function(response) { ... });

getRoute

  • points
  • [options]
  • callback
)

Defined in src\tomtom\services\NavKitRoutingService.js:23

Gets routing information for the specified points.

Parameters:

  • points Array

    An array of L.LatLng objects.

  • [options] Object optional

    Additional options to pass to the routing call.

    • [avoidTraffic=false] Boolean optional

      Indicates if traffic information is used to plan the route. Only relevant for routeType of Quickest and AvoidMotorway. When includeTraffic is set to false this flag is ignored.

    • [includeTraffic=false] Boolean optional

      Indicates if traffic information is used to describe delays in the route. Not relevant where routeType is Walk or Bicycle.

    • [language=en] String optional

      The two character ISO code of the output language to use for the route instructions. Supported languages are ca, cs, da, de, en, es, fi, fr, hu, it, nl, no, pl, pt, sk, sv, tr.

    • [day=today] String optional

      The day of departure. Valid options are today, tomorrow, monday, tuesday, wednesday, thursday, friday, saturday, sunday.

    • [time=now] String optional

      Time of departure, either now or the number of minutes since midnight. Note that now (default value) is only valid when day=today (default).

    • [trafficModelId] String optional

      Traffic model ID to use in the route calculation. The Traffic Model ID can be obtained from the Viewport service. If this is omitted, then the most recent ID will be used.

    • [avoidTolls=false] Boolean optional

      Indicates whether toll roads have to be avoided during route planning.

    • [includeInstructions=true] Boolean optional

      Indicates if turn-by-turn instructions should be returned in the response.

    • [projection=EPSG4326] String optional

      The projection used to specify the coordinates in the request and response. Supported values are EPSG900913 and EPSG4326.

  • callback Function

    A function to be called when the routing call returns a response.

Example:

var points = [ [ 43.20846, -71.53686 ], [ 42.35779, -71.05976 ] ]; var service = new tomtom.services.RoutingService("API Key"); service.getRoute(points, function(response) { ... });
